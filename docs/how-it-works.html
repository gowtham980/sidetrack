sidetrack — How it works

sidetrack is a thin local-first CLI over git worktrees. You issue everyday commands like add, go, list, and rm; it creates a clean sibling layout and maps each action to the underlying git worktree operations.

Example session

cd ~/code/myapp
sidetrack add fix/login          # creates ~/code/myapp-worktrees/fix-login
sidetrack go fix/login           # jump into that checkout
# ... fix, commit, push ...
sidetrack status                 # dirty/clean across all trees
sidetrack rm fix/login -d -y     # remove worktree (+ optional branch)
sidetrack cleanup --yes          # prune merged leftovers
